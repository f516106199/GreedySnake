# GRADY SNAKE  
A tiny Console grady snake game released by c++ .  
platform :windows.  

## build  
``` bash 
cd path/to/gradysnake
mkdir build 
cd build 
cmake -G "Unix Makefiles" ..
make .
```
or you can cd to bin and execute the binary file .  

## operation   
using 'space' to chosing options .  
using ``` w a s d ``` for moving . and ``` space ``` for pausing.  
enter esc to exit or save the data .
